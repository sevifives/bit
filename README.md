bit
===

A multi-repository tool for git written in Javascript for NodeJS/NPM

To install:

npm install -g bit


To use in your git repo:

- bit init

- bit add repo_name ~/real/path/to/repo_name

- bit status 
-> # root git status
-> # repo_name git status

- bit on repo_name status
-> # repo_name git status
